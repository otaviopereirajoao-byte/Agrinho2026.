# 🌱 Agro Forte

Projeto simples para monitoramento agrícola.

## Funcionalidades

- Exibição de temperatura
- Exibição de umidade
- Atualização dinâmica dos dados
- Interface simples e responsiva

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Objetivo

Demonstrar como a tecnologia pode auxiliar no acompanhamento das condições do campo, fortalecendo a produção agrícola.
